# 自キャラ管理ツール

立ち絵、基本情報、追加情報をまとめて管理できる静的Webアプリです。

## 公開方法

このフォルダ内の以下3ファイルを、静的サイトとして公開できるサービスにアップロードします。

- `index.html`
- `styles.css`
- `app.js`

サーバー側のプログラムやデータベースは不要です。

## データ保存について

キャラデータは、利用者それぞれのブラウザ内 `localStorage` に保存されます。

- サーバーには保存されません
- ログインは不要です
- 同じ端末・同じブラウザならデータが残ります
- ブラウザデータを削除すると消える場合があります
- 別端末へ移す場合は、JSONの「書き出し」「読み込み」を使います

## 主な機能

- 複数キャラ管理
- 立ち絵画像の登録
- 基本情報の編集・項目追加
- イメージカラー設定
- 大項目・小項目形式の追加情報
- 小項目への画像追加
- ランダム質問・ランダム項目
- ライト / ダークモード
- JSONバックアップ書き出し / 読み込み

## 注意

画像もブラウザ内に保存するため、大きな画像を大量に登録するとブラウザの保存容量に達する可能性があります。
